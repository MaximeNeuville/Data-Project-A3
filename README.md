Afin de faire fonctionner ce programme il faudra lancer le fichier "main.py" et l'exécuter avec la version 3 de Python.
Cependant, si une erreur arrive, il est possible de lancer séparément les scripts en suivant ces étapes :
	- Pour lancer le plus court chemin avec le recuit simulé, il suffit d'ouvrir le fichier "Pcc_recuit.py" et de dé-commenter le main ainsi que l'import de dsg et tout simplement l'exécuter.
	- Pour lancer le plus court chemin avec le 2opt, il faut faire la même chose avec le fichier "ResolverAlgo.py"
	- Pour lancer le plus court chemin avec une ou deux contraintes il suffit de faire la même chose avec les fichiers "pcc_recuitC1.py" et "pcc_recuitC2.py" respectivement.
